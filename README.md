## 💻 Projeto

O projeto **Jenna Ortega History**, possui a proposta de apresentar aos novos fãs da atriz Jenna Ortega, mostrando toda sua trajetória, de como se tornou uma artista de sucesso na área cinematográfica e televisiva. Nesse contexto, será utilizado o modelo de site chamado “Blog” que significa ser um site informativo, onde os conteúdos são apresentados em ordem cronológica inversa, ou seja, destaca as publicações mais recentes. Além disso, terá a parte da biografia da estrela Jenna Ortega.

## ✅ Tecnologias utilizadas

- HTML
- CSS
- [AOS Animate](https://michalsnik.github.io/aos/)

## 📚 Conhecimentos abordados

- [x] Carregamento de video no background
- [x] Criação do sistema `sliders` no CSS
- [x] Animações com a biblioteca [AOS](https://michalsnik.github.io/aos/)
- [x] Criação da página de filmes
- [x] Efeitos com a propriedade `transform` do css
